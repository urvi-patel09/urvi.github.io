What is CDX Coin?

CDX is a peer-to-peer Internet currency that enables instant, near-zero cost payments to anyone in the world. CDX is an open source, global payment network that is fully decentralized without any central authorities. Mathematics secures the network and empowers individuals to control their own finances. CDX features faster transaction confirmation times and improved storage efficiency than the leading math-based currency. With substantial industry support, trade volume and liquidity, CDX is a proven medium of commerce complementary to Bitcoin.

What is CredX Coin (CDX)?

CredX Coin (CDX) is a digital peer to peer network that aims to implement a use case specific blockchain, designed to efficiently handle one specific function: the transfer of assets from one party to another. Built on a fork of the Bitcoin code, CredX Coin (CDX) was launched 2nd October 2021, and is a truly open source project (no ICO). It focuses on building a useful technology, with a strong and growing community.

Does community members have any CDX?

No there are no coins distributed to any of the developers or to any initial community members. Each and every one will have to go through the block stacking system to mine/get new coins. It will be distributed by the pre defined rules set in to the MasterNodes and Master VMs. Anyone can become block stackig member via contributing to it and start adding new coins to circulating supply.

How CredX Blockchain works?

A fork of the bitcoin code, To help promote Green Revolution there are some major changes done in CredX Blockchain. It features the issuance schedule (block reward of 0.00000001 CDX) and Block time (1 minute) and MasterNodes. Its coins generation system (mining system) is dual layered managed via MasterNodes and Master VM(Virtual Machine) to manage decentralised behaviour of Block Stacking. It is fully explained in WhitePaper Docs.

How there is no carbon footprint of CDX?

To attain decentralized etiquette we have formed a MasterNode and MasterVM concept. With the code of Bitcoin and Ethereum's Stacking Concept. So because there will be limited MasterNodes, there will be not be power usage in mining. And hence it is Green Revolution friendly coin with almost no carbon footprint.
